# Order Flow Terminal v4

Real-time cryptocurrency order flow analysis terminal with advanced alert system.

## Features

- 🧽 Absorption Detection
- 💪 Strong Buy/Sell Pressure Alerts
- 🧱 Buy/Sell Wall Formation Detection
- 🐋 Whale Trading Activity Monitoring
- 🏛 Institutional Trade Detection
- ⚠️ Spoofing Detection
- 🎨 Color-Coded Alerts (Green = Bullish, Red = Bearish)
- 🔊 Voice Announcements
- 📱 Telegram & WhatsApp Notifications

## Deployment

This project is configured for automatic deployment to Vercel.

### Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3011](http://localhost:3011)

### Production

Push to GitHub main branch → Automatic deployment to Vercel

## Live Demo

Access the latest version at your Vercel deployment URL.
